## VAE Demo

This repository includes some demo VAE models.

Datasets:

* `dataset1`: [MNIST](http://yann.lecun.com/exdb/mnist/)

Models:

* `model3`: VAE | [Auto-Encoding Variational Bayes](https://arxiv.org/abs/1312.6114)
